# Telega.in Chrome Extension

MVP-расширение для Chrome, которое читает DOM текущей страницы `telega.in`, извлекает данные каналов и экспортирует их в `JSON` или `CSV`.

## Что умеет

- Парсить текущую карточку канала.
- Парсить все видимые карточки на странице каталога.
- Фильтровать результаты по городу и типу (`telegram` / `max`) прямо в popup.
- Экспортировать результат в `JSON` и `CSV`.

## Структура

- `manifest.json` - конфигурация extension.
- `popup.html` / `popup.js` - интерфейс popup и экспорт.
- `content.js` - чтение DOM и парсинг страницы.
- `styles.css` - стили popup.

## Установка

1. Откройте `chrome://extensions`.
2. Включите `Developer mode`.
3. Нажмите `Load unpacked`.
4. Выберите папку:

```text
C:\Users\Und3rd06\telegain-extension
```

## Использование

1. Откройте страницу `https://telega.in/...`.
2. Нажмите на иконку расширения.
3. При необходимости укажите `Город` и `Тип`.
4. Нажмите `Parse current page`.
5. Используйте `Export JSON` или `Export CSV`.

## Ограничения

- Расширение читает только то, что уже отрисовано на странице.
- Для каталога парсятся только видимые карточки текущей страницы.
- Если `telega.in` поменяет DOM-структуру, эвристики в `content.js` придется обновить.
